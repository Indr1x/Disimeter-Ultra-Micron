Для корректной работы функции авто-определения порта при автозапуске программы, ей необходимы
права Администратора ПК.
Для этого на некоторых версиях Windows, таких как 8 и 8.1 необходимо:

1.Включить учетную запись администратора
  Панель управления -> Администрирование -> Управление компьютером -> Локальные пользователи и группы
  -> Пользователи -> Администратор.
  Там надо снять флажок "Отключить учетную запись"
  Потом через правую кнопку мыши "задать пароль" любой какой вы захотите
  И потом снова через правую кнопку мыши переименовать учетную запись в "Administrator"

2.Положить прилагаемый файл Micron.cmd в папку с программой.

3.Убедится, что программа лежит в папках не содержащих русских букв.

4.Снять и установать флажок автозапуск.

5.После первого автозапуска, CMD файл скорее всего потребует ввода пароля администратора.


PS.Пути в micron.cmd будут автоматически исправлены програмой при необходимости.
